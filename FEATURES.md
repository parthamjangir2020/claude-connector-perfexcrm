# Features

## Database Operations

| Feature | Description |
|---------|-------------|
| Execute SQL | Run SELECT, INSERT, UPDATE, DELETE queries |
| List Tables | View all database tables with row counts |
| Describe Table | Get schema with column types and keys |
| Backup Table | Create backups before modifications |

## Lead Management

- List leads with filters (status, source, assigned staff)
- Create new leads with custom fields
- Update lead information
- Convert leads to clients
- Assign leads to staff members

## Client Management

- List clients/companies
- Create new clients
- Update client information
- Manage client contacts
- Access client custom fields

## Project Management

- List projects by client or status
- Create new projects
- Update project details
- Manage project milestones
- Track timesheets

## Task Management

- List tasks across all projects
- Create tasks with assignees
- Update task status and priority
- Manage task timers
- Add task comments

## Invoice Management

- List invoices by status
- Create new invoices
- Update invoice items
- Track payments
- Generate PDF triggers

## Internal Communication

| Feature | Description |
|---------|-------------|
| Add Notes | Add internal notes to any entity |
| Activity Log | View recent activity for any record |
| Email Templates | Trigger emails using CRM templates |
| Audit Trail | Log custom actions for tracking |

## Module Discovery

- List all installed Perfex modules
- Get module details and capabilities
- Access module-specific database tables
- Execute module-specific actions

## Staff Operations

- List staff members
- View staff roles and permissions
- Get staff activity reports
