# Version History

## [1.0.0] - 2024-12-10

### Added
- Initial release of Claude Connector for Perfex CRM
- MCP Server with TypeScript implementation
- Database tools (execute SQL, list tables, describe schema)
- Lead management tools (list, create, update, convert)
- Client management tools (list, create, update)
- Project management tools (list, create, update)
- Task management tools (list, create, update, assign)
- Invoice management tools (list, create)
- Communication tools (notes, activity log)
- Module discovery (list installed modules, module tables)
- Perfex CRM admin module for settings
- Claude Desktop configuration example
- Complete documentation (README, FEATURES, SETUP, MODULE_OVERVIEW)

- @modelcontextprotocol/sdk integration

- Environment-based configuration
